# Samizdat

*самиздат* — "self-published"

A distraction-free writing app for Nostr. Write long-form content and publish it to the world without permission, without gatekeepers, without censorship.

Soviet dissidents hand-copied forbidden literature and passed it from reader to reader. Nostr does the same thing at the speed of light.

## What it is

A beautiful, minimal editor that publishes NIP-23 long-form content to Nostr relays. Nothing else.

- Open the app
- Write
- Publish

No feed. No notifications. No algorithm. Just your words.

## Stack

- React + TypeScript + Vite
- Tiptap (rich text editor)
- nostr-tools (event signing & relay communication)
- Source Serif 4 + Inter typography
- PWA (works on phone + desktop)
